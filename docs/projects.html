AFSKModem

An error-correcting AFSK digital radio protocol designed to enable digital data to be transmitted and received by analog-only radios with no external hardware besides a personal computer.

https://lavajuno.github.io/afskmodem

The above is the initial implementation of afskmodem in Python. I also implemented it in Java (linked below). Both versions are fully compatible with each other.

https://lavajuno.github.io/jfskmodem

LucidJSON

A small, easy-to-use JSON serialization/deserialization library for Java with a LL parser backend.

https://github.com/lavajuno/lucidjson

MirrorLog

A customizable cross-platform log server designed for use by Clarkson University's open source mirror.

https://lavajuno.github.io/mirrorlog

Mount Fsuvius

A web application and API allowing users to create, edit, and delete account entries to track their balance of FSU, to be used by Clarkson Open Source Institute in their labs.

https://lavajuno.github.io/fsuvius
